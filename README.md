# Online FIR Complaint Management System
## Overview
Online FIR Complaint Management System is a web-based application developed to simplify the process of filing and managing FIR complaints digitally. The system helps users submit complaints online while allowing administrators to monitor, verify, and update complaint status efficiently.
## Features
### User Module
- User Registration
- Secure Login
- File FIR Complaints
- Upload Proof/Documents
- Track FIR Status

### Admin Module
- Accept FIR Requests
- Update FIR Details
- Manage Complaint Records
- Verify Submitted Proofs

## Modules
- Login Module
- Register Module
- Complaint FIR Module
- FIR Status Module
- Accept FIR Module
- Update FIR Module
- Proof Upload Module

## Technologies Used
- Front End: HTML, Bootstrap, jQuery
- Back End: PHP
- Database: MySQL
- Server: XAMPP
- Operating System: Windows

## Hardware Requirements
- Processor: Intel Core i3
- RAM: 4 GB
- Hard Disk: 25 GB

## Purpose
The main objective of this project is to reduce manual paperwork and provide a faster, secure, and user-friendly platform for managing FIR complaints online.
